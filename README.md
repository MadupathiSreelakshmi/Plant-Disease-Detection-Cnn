# 🌿 Plant Disease Detection using CNN

## 📌 Overview
This project is a deep learning-based Plant Disease Detection System developed using TensorFlow and Keras. The model classifies plant leaf diseases from images using Convolutional Neural Networks (CNN) and helps in early disease identification for improved crop health and productivity.

---

## 🚀 Features
- Plant leaf disease classification using CNN
- Trained on 87,000+ leaf images
- Supports 38 disease categories
- Real-time disease prediction
- User-friendly web interface for image upload and prediction

---

## 🛠️ Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib
- Flask

---

## ⚙️ Model Performance
- Training Accuracy: 95%
- Testing Accuracy: 92%
- Applied image preprocessing and early stopping techniques for improved model performance

---

## 📊 Project Workflow
1. Upload plant leaf image
2. Image preprocessing and resizing
3. CNN model prediction
4. Display predicted disease result
